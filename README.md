# Saudi Unemployment Rate Forecasting System


> An AI-powered system that forecasts Saudi Arabia's national unemployment rate using historical quarterly data from DataSaudi, aligned with Vision 2030 targets.

---

## Project Overview

This system trains and compares three forecasting models (Linear Regression, Random Forest, SARIMA) and deploys the best-performing model as an interactive **Streamlit dashboard** hosted on Streamlit Cloud.

| Item | Detail |
| **Best Model** | SARIMA (1,1,1)(1,1,1,4) |
| **MAE** | 0.4551 percentage points |
| **R²** | 0.4524 |
| **Data Source** | DataSaudi (Q2 2016 – Q4 2025) |
| **Forecast** | 2026-Q1: 7.91% \| 2026-Q2: 8.71% |

##  How to Run — Step by Step

### Step 1: Set Up Google Drive

1. Go to [Google Drive](https://drive.google.com)
2. Create this exact folder structure:
   ```
   MyDrive/
    unemployment_project/
        raw_data/
            Unemployment-Rate-by-Gender.csv
        streamlit_app/
   
3. Download the dataset from [DataSaudi](https://datasaudi.sa) Search for: Unemployment Rate by Gender
Upload the CSV file to the raw_data folder

### Step 2: Run the Training Notebook

1. Open [Google Colab](https://colab.research.google.com)
2. Upload `Copy_of_AISproject.ipynb` or open it from GitHub
3. Click **Runtime then Run all** 
4. When prompted, **authorize Google Drive access** and click "Connect to Google Drive"
5. Wait for all cells to finish — this takes approximately 2–5 minutes
6. After completion, the following files will be saved automatically to your Drive:
   - `unemployment_project/rf_model.pkl`
   - `unemployment_project/sarima_model.pkl`
   - `unemployment_project/scaler.pkl`
   - `unemployment_project/streamlit_app/results.json`
   - `unemployment_project/streamlit_app/monitoring_log.json`

### Step 3: Run the Streamlit Dashboard Locally (Optional)

Live dashboard (no installation required):
https://saudi-unemployment-forecast-hkuptipfttegknmkxdbzqg.streamlit.app

To run locally:

    pip install -r requirements.txt
    streamlit run app.py

Then open: http://localhost:8501

Note: You must copy sarima_model.pkl, results.json, and 
monitoring_log.json into the same folder as app.py before 
running locally.

##  Retraining the Model

When new quarterly data is released by DataSaudi:

1. Download the updated CSV file from DataSaudi
2. Replace the file in `MyDrive/unemployment_project/raw_data/`
3. Re-run all cells in the Colab notebook
4. The new model files and `results.json` will be saved automatically
5. Push the updated `results.json` and `monitoring_log.json` to GitHub
6. Streamlit Cloud will redeploy automatically
